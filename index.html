p5.playground

A workshop of small p5.js sketches — generative art, simulations, games, and a new learning wing where software evolves and visibly gets better. Hover any card to run it in place.

Hover previews run the real sketch in a sandboxed frame and stop the moment you leave. Sketches that use media (sound, video, images, microphone) need to be served over HTTP — run python -m http.server from the repo root.
